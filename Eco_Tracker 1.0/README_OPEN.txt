PASOS RÁPIDOS (Windows, Visual Studio Code):
1) Abrir esta carpeta en VS Code (Archivo -> Abrir carpeta).
2) Presionar F5 y elegir "Eco_Tracker ▶ Auto-Run (Windows)".
   - VS Code usará autorun.py para crear la venv, instalar dependencias y ejecutar.
Si no aparece la opción, ve a .vscode/launch.json y elige esa configuración.
